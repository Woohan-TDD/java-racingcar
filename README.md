# java-racingcar
자동차 경주 게임 미션 저장소

# 기능 요구사항
## 시나리오
- 이름을 입력받는다.
    - 이름은 1자 이상, 5자 이하로 구성된다.
    - 이름은 빈 문자열로만 이루어질 수 없다.
    - 문자 앞, 뒤로 오는 공백 문자는 무시한다.
    - 2 ~ 10명 사이의 이름을 입력받아야 한다.
- 시도 횟수를 입력받는다.
    - 게임은 1 ~ 10 사이의 정수이다.
- 시도 횟수 만큼 결과를 출력한다.
- 최종 우승자를 출력한다.
